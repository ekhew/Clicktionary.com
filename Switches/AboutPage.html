About

What is Click*tionary.com?

Click*tionary.com is a catalog/guide I created that focuses mainly on mechanical keyboard switches. It organizes most, if not all, of the switches that exist into one convenient place. Here, you can browse through the catalog and learn about individual switches, or read through the guides and learn about switches in general. I was motivated to create this website because when I was still relatively new to the mechanical keyboard hobby, I felt overwhelmed by all the different switches that I saw while wandering through r/mechanicalkeyboards. Hopefully this website would help get you started on finding that perfect switch for your first build, or that next switch that you would like to try out (if its in stock).
